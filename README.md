# AcuarelaSoft

El arte de crear software.

AcuarelaSoft is a custom software studio based in Mexico City, focused on building scalable digital products for teams that need more than code: clear product thinking, solid architecture, and polished UX.

This repository powers the public AcuarelaSoft website and project intake flow, built with Laravel, Livewire, and Tailwind.

## What We Build

- Web apps and business platforms (CRM, ERP-like workflows, internal operations software)
- Public and private APIs (REST, GraphQL, SOAP integrations)
- Mobile and desktop app projects
- Legacy modernization and migration
- Ongoing app maintenance, performance, and reliability improvements
- Cloud and server operations support

## How We Work

1. Discovery and technical alignment
2. Clear module-based scope definition
3. Iterative delivery with continuous feedback
4. Production hardening and support

The app includes a module-based intake estimator that helps translate business requirements into an initial effort and budget tier estimate.

## Core Product Features in This App

- Bilingual site experience (Spanish/English localized routes)
- Service catalog with dedicated pages and structured data (SEO-first)
- Contact flow with anti-spam protections (Cloudflare Turnstile in production)
- Livewire-powered project intake form with:
  - module search and filtering
  - weighted complexity scoring
  - estimate size/budget tier generation
  - persistence of submissions
  - email confirmation workflow
- Sitemap and robots endpoints

## Engineering Principles

- Product and UX first
- Strong backend fundamentals
- Secure-by-default forms and workflows
- SEO and performance as baseline concerns
- Incremental, maintainable delivery over one-off hacks
